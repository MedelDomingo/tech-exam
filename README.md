# Landing Page

React Vite landing page.

## Packages

- [@fortawesome/fontawesome-free](https://fontawesome.com/) - Free SVG icons.
- [gh-pages](https://www.npmjs.com/package/gh-pages) - Deploys to GitHub Pages.
- [react-slick](https://react-slick.neostack.com/) - Carousel slider component.

## Clone
```bash
git clone git@github.com:MedelDomingo/tech-exam.git
git to directory
```

## Run it on dev
```bash
npm install
npm run dev
